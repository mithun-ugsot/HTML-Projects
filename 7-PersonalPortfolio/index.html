About Me

I am a Finnish-American software engineer, best known for creating and maintaining the Linux kernel and initiating the development of Git.

My philosophy centers around open-source software and collaborative development. I believe in "Talk is cheap. Show me the code."

Major Projects

Linux Kernel

Role: Creator & Principal Developer

Started: August 1991

Description
A free and open-source, Unix-like operating system kernel that powers millions of devices worldwide.
Impact
Powers supercomputers, Android devices, servers, and embedded systems globally.
Technology
Written primarily in C and Assembly language

Initial announcement message:

Hello everybody out there using minix -
I'm doing a (free) operating system (just a hobby, 
won't be big and professional like gnu) for 386(486) AT clones.

Learn more: kernel.org

Git

Role: Creator & Original Developer

Started: April 2005

Description
A distributed version control system designed to handle everything from small to very large projects.
Impact
Industry standard for version control, used by millions of developers worldwide.
Technology
Written in C, Shell, and Perl
Why I created Git

I needed a version control system for Linux kernel development that was fast, distributed, and could handle large projects efficiently. Existing solutions didn't meet these requirements.

Learn more: git-scm.com
