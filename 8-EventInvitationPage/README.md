# Event Invitation Page Project - HTML Fundamentals

## 1. Problem Statement

Create a simple event invitation page for "TechConnect 2026 - Annual Technology Summit" using **only HTML** (no CSS or JavaScript). The page demonstrates essential HTML concepts **not covered in the Portfolio project**, including images, tables, embedded YouTube videos, Google Maps, and interactive FAQ sections. The focus is on multimedia and embedded content to complement existing projects.

## 3. Expected Outcomes

1. **Images:** Master `<img>` element with `src`, `alt`, `width`, and `height` attributes for accessibility
2. **Semantic Media:** Use `<figure>` and `<figcaption>` to group images with captions semantically
3. **Tables:** Create structured data tables with `<table>`, `<caption>`, `<thead>`, `<tbody>`, `<tr>`, `<th>`, `<td>`
4. **YouTube Embed:** Embed YouTube videos using `<iframe>` with proper attributes like `allowfullscreen`
5. **Google Maps:** Integrate interactive Google Maps using `<iframe>` with `loading="lazy"` for performance
6. **Interactive FAQ:** Implement collapsible Q&A sections using `<details>` and `<summary>` elements
7. **Blockquotes:** Use `<blockquote>` with proper citation using `<cite>` and `<footer>` elements
8. **Contact Information:** Apply `<address>` element for venue location and contact details
9. **Data Semantics:** Use `<time>` for dates and `<data>` for prices with machine-readable values
10. **Page Structure:** Organize content with semantic HTML5 elements (`<header>`, `<main>`, `<section>`, `<footer>`)
